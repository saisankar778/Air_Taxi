# Sky Works — Next-Gen Air Mobility

**Sky Works** is a conceptual electric vertical take-off and landing (eVTOL) air taxi system designed for future urban mobility. This project explores the integration of autonomous flight, sustainable energy, and smart city infrastructure to redefine urban transportation.

## 🚀 Features

- **Autonomous Flight System**: AI-powered navigation and collision avoidance
- **Electric Propulsion**: Zero-emission vertical take-off and landing
- **Smart City Integration**: Seamless connectivity with urban infrastructure
- **Modular Design**: Configurable cabin for passengers or cargo
- **Advanced Aerodynamics**: Optimized for urban flight corridors

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **3D Graphics**: Three.js, WebGL
- **Icons**: Phosphor Icons
- **Fonts**: Space Grotesk, Syne Mono, Rajdhani
- **Deployment**: Vercel

## ⚙️ Setup & Deployment

### Prerequisites

- Node.js (for Vercel CLI)
- Vercel CLI (`npm install -g vercel`)

### Deployment

1. **Install Vercel CLI**:
   ```bash
   npm install -g vercel
   ```

2. **Deploy to Vercel**:
   ```bash
   vercel --prod
   ```

3. **Access the Application**:
   Once deployed, Vercel will provide a URL to access the live application.

## 📂 Local Development

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd Air_Taxi
   ```

2. **Open in browser**:
   Open `index.html` in your web browser or use a local development server:
   ```bash
   # Using Python
   python3 -m http.server 8000
   
   # Using Node.js
   npx serve .
   ```
